# CLI & Interactive Menu Guide

## Interactive Menu Hierarchy
- **Analyze Repository**
  - `Quick`: Top-level metrics only.
  - `Detailed`: Full scan including file tree.
- **Compare Repositories**: Side-by-side analysis.
- **Settings**: 
  - Theme customization (Neon/Dark/True Black).
  - GitHub Token configuration.

## Command Flags
| Flag | Description |
|------|-------------|
| `--summary` | Skips TUI and prints a 5-line text summary. |
| `--export` | Auto-exports result to a specified format. |
